## Project Features
### 首頁
* 點擊播放=>跳轉到播放影片頁面
* 點擊收藏=>把影片加入到收藏頁
- 影片資訊：
    - 圖片
    - 標題
    - 描述
- 頁籤：
    - 總計列出100部影片
    - 每個頁面呈現12部影片
    - 切換=>進行不同影片觀看與收藏
    - 根據影片數量進行動態生成

### 收藏頁
* 呈現收藏影片
* 點擊播放=>跳轉到播放影片頁面
* 點擊刪除最愛=>把收藏影片刪除
* 重整頁面收藏資料不會消失
- 頁籤：
    - 每個頁面呈現12部影片
    - 根據收藏影片數量進行動態生成
### 播放頁
* 可以播放/暫停影片
* 調整播放速度

* axios（Post/Get API）

* Vue Router （頁面動態跳轉）
